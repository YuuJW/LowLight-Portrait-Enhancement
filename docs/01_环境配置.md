## 1. C++ 开发环境

### 1.1 安装 Visual Studio 2022

1. 下载地址：https://visualstudio.microsoft.com/
2. 选择 **Community 版本**（免费）
3. 安装时勾选以下组件：
   - ✅ 使用 C++ 的桌面开发
   - ✅ Windows 10/11 SDK
   - ✅ CMake 工具（可选，推荐单独安装）

### 1.2 安装 CMake

**方法1：使用 winget（推荐）**
```powershell
winget install Kitware.CMake
```

**方法2：手动下载**
- 下载地址：https://cmake.org/download/
- 选择 Windows x64 Installer
- 安装时勾选 "Add CMake to system PATH"

**验证安装**
```powershell
cmake --version
# 应输出: cmake version 3.x.x
```

### 1.3 安装 Git

```powershell
winget install Git.Git
```

或从 https://git-scm.com/download/win 下载安装

## 2. Python 环境

### 2.1 安装 Miniconda

1. 下载地址：https://docs.conda.io/en/latest/miniconda.html
2. 选择 Windows 64-bit 版本
3. 安装时勾选 "Add Miniconda to PATH"

### 2.2 创建项目虚拟环境

```powershell
# 创建环境
conda create -n isp python=3.10

# 激活环境
conda activate isp
```

### 2.3 安装 PyTorch

**CUDA 11.8 版本（推荐，需要 NVIDIA 显卡）**
```powershell
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

**CPU 版本（无显卡）**
```powershell
pip install torch torchvision torchaudio
```

### 2.4 安装其他 Python 依赖

```powershell
# 基础图像处理
pip install opencv-python numpy matplotlib scikit-image

# RAW 图像处理
pip install rawpy imageio

# 模型部署
pip install onnx onnxruntime-gpu

# 图像质量评估
pip install lpips

# 训练框架（可选）
pip install basicsr
```

### 2.5 完整 requirements.txt

创建 `requirements.txt` 文件：

```txt
torch>=2.0.0
torchvision>=0.15.0
torchaudio>=2.0.0
opencv-python>=4.8.0
numpy>=1.24.0
matplotlib>=3.7.0
scikit-image>=0.21.0
rawpy>=0.18.0
imageio>=2.31.0
onnx>=1.14.0
onnxruntime-gpu>=1.15.0
lpips>=0.1.4
tqdm>=4.65.0
pyyaml>=6.0
```

安装：
```powershell
pip install -r requirements.txt
```

## 3. OpenCV C++ 配置

### 方法1：使用 vcpkg（推荐）

```powershell
# 克隆 vcpkg
git clone https://github.com/Microsoft/vcpkg.git
cd vcpkg

# 初始化
.\bootstrap-vcpkg.bat

# 安装 OpenCV
.\vcpkg install opencv4:x64-windows

# 集成到 Visual Studio
.\vcpkg integrate install
```

**在 CMakeLists.txt 中使用：**
```cmake
find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})
target_link_libraries(your_project ${OpenCV_LIBS})
```

### 方法2：下载预编译版

1. 下载地址：https://opencv.org/releases/
2. 选择 Windows 版本下载
3. 解压到 `D:\libs\opencv\`
4. 添加环境变量：`D:\libs\opencv\build\x64\vc16\bin`

**CMakeLists.txt 配置：**
```cmake
set(OpenCV_DIR "D:/libs/opencv/build")
find_package(OpenCV REQUIRED)
```

## 4. ONNX Runtime 配置

### 4.1 下载预编译库

1. 访问：https://github.com/microsoft/onnxruntime/releases
2. 下载：`onnxruntime-win-x64-gpu-1.x.x.zip`
3. 解压到：`D:\libs\onnxruntime\`

### 4.2 目录结构

```
D:\libs\onnxruntime\
├── include\
│   └── onnxruntime_cxx_api.h
├── lib\
│   ├── onnxruntime.lib
│   └── onnxruntime_providers_cuda.lib
└── bin\
    ├── onnxruntime.dll
    └── onnxruntime_providers_cuda.dll
```

### 4.3 CMakeLists.txt 配置

```cmake
# ONNX Runtime 配置
set(ONNXRUNTIME_DIR "D:/libs/onnxruntime")
include_directories(${ONNXRUNTIME_DIR}/include)
link_directories(${ONNXRUNTIME_DIR}/lib)

target_link_libraries(your_project onnxruntime)
```

### 4.4 添加环境变量

将 `D:\libs\onnxruntime\bin` 添加到系统 PATH 环境变量

## 6. 环境验证

### 6.1 验证 Python 环境

```python
# test_env.py
import torch
import cv2
import numpy as np

print(f"PyTorch version: {torch.__version__}")
print(f"CUDA available: {torch.cuda.is_available()}")
if torch.cuda.is_available():
    print(f"CUDA version: {torch.version.cuda}")
    print(f"GPU: {torch.cuda.get_device_name(0)}")

print(f"OpenCV version: {cv2.__version__}")
print(f"NumPy version: {np.__version__}")

# 测试 ONNX Runtime
try:
    import onnxruntime as ort
    print(f"ONNX Runtime version: {ort.__version__}")
    print(f"Available providers: {ort.get_available_providers()}")
except ImportError:
    print("ONNX Runtime not installed")
```

运行：
```powershell
conda activate isp
python test_env.py
```

### 6.2 验证 C++ 环境

创建测试项目 `test_cpp/`：

**CMakeLists.txt**
```cmake
cmake_minimum_required(VERSION 3.15)
project(TestEnv)

set(CMAKE_CXX_STANDARD 17)

find_package(OpenCV REQUIRED)

add_executable(test_env main.cpp)
target_link_libraries(test_env ${OpenCV_LIBS})
```

**main.cpp**
```cpp
#include <opencv2/opencv.hpp>
#include <iostream>

int main() {
    std::cout << "OpenCV version: " << CV_VERSION << std::endl;

    // 测试图像读写
    cv::Mat img = cv::Mat::zeros(100, 100, CV_8UC3);
    cv::circle(img, cv::Point(50, 50), 30, cv::Scalar(0, 255, 0), -1);
    cv::imwrite("test_output.png", img);

    std::cout << "Test passed!" << std::endl;
    return 0;
}
```

编译运行：
```powershell
mkdir build && cd build
cmake ..
cmake --build . --config Release
.\Release\test_env.exe
```

## 7. 常见问题

### Q1: CUDA 不可用

**症状**：`torch.cuda.is_available()` 返回 False

**解决**：
1. 确认安装了 NVIDIA 驱动
2. 确认安装了正确版本的 CUDA Toolkit
3. 重新安装 PyTorch：
```powershell
pip uninstall torch torchvision torchaudio
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Q2: CMake 找不到 OpenCV

**解决**：
```cmake
# 在 CMakeLists.txt 中明确指定路径
set(OpenCV_DIR "D:/libs/opencv/build")
find_package(OpenCV REQUIRED)
```

### Q3: DLL 找不到

**症状**：运行时报错 "xxx.dll not found"

**解决**：
1. 将 DLL 所在目录添加到 PATH 环境变量
2. 或将 DLL 复制到 exe 同目录

### Q4: conda activate 失败

**解决**：
```powershell
# 初始化 conda
conda init powershell
# 重启 PowerShell
```

### Q5: pip 安装超时

**解决**：使用国内镜像
```powershell
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple package_name
```

或配置永久镜像：
```powershell
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```
