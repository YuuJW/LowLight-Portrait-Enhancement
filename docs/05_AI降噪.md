## 1. 图像降噪概述

### 1.1 图像噪声类型

| 噪声类型 | 来源 | 特点 |
|---------|------|------|
| **高斯噪声** | 传感器热噪声 | 符合高斯分布，与信号无关 |
| **泊松噪声** | 光子散粒噪声 | 与信号强度相关，暗部更明显 |
| **椒盐噪声** | 传输错误 | 随机黑白点 |
| **条纹噪声** | 读出电路 | 水平/垂直条纹 |

### 1.2 噪声模型

**加性高斯白噪声（AWGN）**：
```
y = x + n
其中 n ~ N(0, σ²)
```

**泊松-高斯噪声（更真实）**：
```
y = x + √x · n_p + n_g
其中 n_p ~ N(0, 1), n_g ~ N(0, σ²)
```

### 1.3 降噪的挑战

- 去除噪声的同时保留细节
- 边缘保持（不模糊边缘）
- 计算效率（实时性要求）

## 2. 传统降噪方法

### 2.1 高斯滤波

```python
import cv2

# 高斯滤波
denoised = cv2.GaussianBlur(noisy, (5, 5), sigma)
```

```python

```

**优点**：保留边缘
**缺点**：计算慢，参数敏感


**优点**：效果好
**缺点**：计算非常慢

### 2.4 BM3D

- 当前最好的传统方法之一
- 基于块匹配和 3D 协同滤波
- 计算复杂度高

## 3. AI 降噪原理

### 3.1 为什么用深度学习？

| 传统方法 | AI 方法 |
|---------|--------|
| 手工设计先验 | 从数据学习先验 |
| 参数有限 | 大量可学习参数 |
| 难以处理复杂噪声 | 适应多种噪声模式 |
| 速度慢（NLM、BM3D） | GPU 加速，可实时 |

### 3.2 残差学习

**关键思想**：学习噪声而非干净图像

```
传统方法：y → CNN → x̂（直接预测干净图像）

残差学习：y → CNN → n̂ → x̂ = y - n̂（预测噪声，然后相减）
```

**优点**：
- 更容易学习（噪声比图像简单）
- 梯度传播更稳定
- 收敛更快

### 3.3 经典模型演进

```
DnCNN (2017)
    ↓
FFDNet (2018) - 支持可变噪声等级
    ↓
RDN (2018) - 残差密集网络
    ↓
NBNet (2021) - 非局部块
    ↓
Restormer (2022) - Transformer 架构
    ↓
NAFNet (2022) - 简单高效
```

## 4. DnCNN 实现

### 4.1 网络结构

```
Input (H×W×C)
    ↓
Conv(3×3, 64) + ReLU
    ↓
[Conv(3×3, 64) + BN + ReLU] × 15  (中间层)
    ↓
Conv(3×3, C)  (输出噪声残差)
    ↓
Output = Input - Residual
```

### 4.2 PyTorch 实现

```python
# models/dncnn.py
import torch
import torch.nn as nn

class DnCNN(nn.Module):
    """
    DnCNN 降噪网络

    Args:
        in_channels: 输入通道数（灰度=1，彩色=3）
        out_channels: 输出通道数
        num_layers: 网络层数（默认17）
        num_features: 特征通道数（默认64）
    """
    def __init__(self, in_channels=1, out_channels=1, num_layers=17, num_features=64):
        super(DnCNN, self).__init__()

        layers = []

        # 第一层：Conv + ReLU
        layers.append(nn.Conv2d(in_channels, num_features, kernel_size=3, padding=1, bias=False))
        layers.append(nn.ReLU(inplace=True))

        # 中间层：Conv + BN + ReLU
        for _ in range(num_layers - 2):
            layers.append(nn.Conv2d(num_features, num_features, kernel_size=3, padding=1, bias=False))
            layers.append(nn.BatchNorm2d(num_features))
            layers.append(nn.ReLU(inplace=True))

        # 最后一层：Conv（输出残差）
        layers.append(nn.Conv2d(num_features, out_channels, kernel_size=3, padding=1, bias=False))

        self.dncnn = nn.Sequential(*layers)

    def forward(self, x):
        # 预测噪声残差
        residual = self.dncnn(x)
        # 残差学习：干净图像 = 噪声图像 - 噪声
        return x - residual


# 测试
if __name__ == '__main__':

    print(f"参数量: {sum(p.numel() for p in model.parameters()) / 1e6:.2f}M")

    x = torch.randn(1, 3, 256, 256)
    y = model(x)
    print(f"输入: {x.shape}, 输出: {y.shape}")
```

### 4.3 改进版本：带噪声等级输入

```python
class DnCNN_NL(nn.Module):
    """
    带噪声等级图的 DnCNN（类似 FFDNet）
    """
    def __init__(self, in_channels=1, out_channels=1, num_layers=17, num_features=64):
        super().__init__()

        # 输入通道 +1（噪声等级图）
        self.first_conv = nn.Conv2d(in_channels + 1, num_features, 3, padding=1, bias=False)
        self.first_relu = nn.ReLU(inplace=True)

        # 中间层
        self.middle_layers = nn.ModuleList()
        for _ in range(num_layers - 2):
            self.middle_layers.append(nn.Sequential(
                nn.Conv2d(num_features, num_features, 3, padding=1, bias=False),
                nn.BatchNorm2d(num_features),
                nn.ReLU(inplace=True)
            ))

        # 最后一层
        self.last_conv = nn.Conv2d(num_features, out_channels, 3, padding=1, bias=False)

    def forward(self, x, noise_level):
        # noise_level: (B, 1, 1, 1) → 扩展为 (B, 1, H, W)
        B, C, H, W = x.shape
        noise_map = noise_level.expand(B, 1, H, W)

        # 拼接噪声等级图
        x_concat = torch.cat([x, noise_map], dim=1)

        # 前向传播
        out = self.first_relu(self.first_conv(x_concat))
        for layer in self.middle_layers:
            out = layer(out)
        residual = self.last_conv(out)

        return x - residual
```

### 5.2 数据增强

```python
# dataset.py
import torch
from torch.utils.data import Dataset
import cv2
import numpy as np
import random

class DenoisingDataset(Dataset):
    def __init__(self, image_paths, patch_size=64, noise_level=25, augment=True):
        self.image_paths = image_paths
        self.patch_size = patch_size
        self.noise_level = noise_level
        self.augment = augment

    def __len__(self):
        return len(self.image_paths)

    def __getitem__(self, idx):
        # 读取图像
        img = cv2.imread(self.image_paths[idx])
        img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
        img = img.astype(np.float32) / 255.0

        # 随机裁剪 patch
        h, w = img.shape[:2]
        top = random.randint(0, h - self.patch_size)
        left = random.randint(0, w - self.patch_size)
        patch = img[top:top+self.patch_size, left:left+self.patch_size]

        # 数据增强
        if self.augment:
            # 随机翻转
            if random.random() > 0.5:
                patch = np.fliplr(patch)
            if random.random() > 0.5:
                patch = np.flipud(patch)
            # 随机旋转 90°
            k = random.randint(0, 3)
            patch = np.rot90(patch, k)

        # 添加噪声
        noise = np.random.randn(*patch.shape).astype(np.float32) * (self.noise_level / 255.0)
        noisy_patch = patch + noise
        noisy_patch = np.clip(noisy_patch, 0, 1)

        # 转换为 tensor
        clean = torch.from_numpy(patch.transpose(2, 0, 1).copy())
        noisy = torch.from_numpy(noisy_patch.transpose(2, 0, 1).copy())

        return noisy, clean
```

### 5.3 训练脚本

```python
# train.py
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader
from models.dncnn import DnCNN
from dataset import DenoisingDataset
import glob
from tqdm import tqdm

def train():
    # 配置
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    batch_size = 32
    num_epochs = 100
    learning_rate = 1e-3
    noise_level = 25

    # 数据集
    train_images = glob.glob('datasets/BSD400/*.png')
    train_dataset = DenoisingDataset(train_images, patch_size=64, noise_level=noise_level)
    train_loader = DataLoader(train_dataset, batch_size=batch_size, shuffle=True, num_workers=4)

    # 模型
    model = DnCNN(in_channels=3, out_channels=3, num_layers=17).to(device)

    # 损失函数和优化器
    criterion = nn.MSELoss()
    optimizer = optim.Adam(model.parameters(), lr=learning_rate)
    scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=30, gamma=0.5)

    # 训练循环
    for epoch in range(num_epochs):
        model.train()
        epoch_loss = 0

        pbar = tqdm(train_loader, desc=f'Epoch {epoch+1}/{num_epochs}')
        for noisy, clean in pbar:
            noisy = noisy.to(device)
            clean = clean.to(device)

            # 前向传播
            output = model(noisy)
            loss = criterion(output, clean)

            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            epoch_loss += loss.item()
            pbar.set_postfix({'loss': loss.item()})

        scheduler.step()

        avg_loss = epoch_loss / len(train_loader)
        print(f'Epoch {epoch+1}, Average Loss: {avg_loss:.6f}')

        # 保存模型
        if (epoch + 1) % 10 == 0:
            torch.save(model.state_dict(), f'checkpoints/dncnn_epoch_{epoch+1}.pth')

    torch.save(model.state_dict(), 'checkpoints/dncnn_final.pth')
    print('Training completed!')


if __name__ == '__main__':
    train()
```

### 6.2 ONNX Runtime 推理（Python）

```python
# inference_onnx.py
import onnxruntime as ort
import numpy as np
import cv2

def denoise_image(image_path, onnx_path):
    # 读取图像
    img = cv2.imread(image_path)
    img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
    img = img.astype(np.float32) / 255.0

    # 转换为 NCHW 格式
    img_tensor = img.transpose(2, 0, 1)[np.newaxis, ...]

    # 创建 ONNX Runtime 会话
    providers = ['CUDAExecutionProvider', 'CPUExecutionProvider']
    session = ort.InferenceSession(onnx_path, providers=providers)

    # 推理
    input_name = session.get_inputs()[0].name
    output = session.run(None, {input_name: img_tensor})[0]

    # 后处理
    output = output.squeeze().transpose(1, 2, 0)
    output = np.clip(output * 255, 0, 255).astype(np.uint8)
    output = cv2.cvtColor(output, cv2.COLOR_RGB2BGR)

    return output


if __name__ == '__main__':
    result = denoise_image('noisy.jpg', 'checkpoints/dncnn.onnx')
    cv2.imwrite('denoised.jpg', result)
```

### 6.3 C++ 推理

```cpp
// denoise_cpp.cpp
#include <onnxruntime_cxx_api.h>
#include <opencv2/opencv.hpp>
#include <vector>

class DnCNNDenoiser {
public:
    DnCNNDenoiser(const std::string& model_path) {
        env_ = Ort::Env(ORT_LOGGING_LEVEL_WARNING, "DnCNN");
        session_options_.SetGraphOptimizationLevel(GraphOptimizationLevel::ORT_ENABLE_ALL);

        // 使用 CUDA（如果可用）
        // OrtCUDAProviderOptions cuda_options;
        // session_options_.AppendExecutionProvider_CUDA(cuda_options);

        session_ = Ort::Session(env_, model_path.c_str(), session_options_);
    }

    cv::Mat denoise(const cv::Mat& input) {
        // 预处理
        cv::Mat img_float;
        input.convertTo(img_float, CV_32FC3, 1.0 / 255.0);
        cv::cvtColor(img_float, img_float, cv::COLOR_BGR2RGB);

        // HWC -> CHW
        std::vector<float> input_tensor(3 * input.rows * input.cols);
        for (int c = 0; c < 3; ++c) {
            for (int h = 0; h < input.rows; ++h) {
                for (int w = 0; w < input.cols; ++w) {
                    input_tensor[c * input.rows * input.cols + h * input.cols + w] =
                        img_float.at<cv::Vec3f>(h, w)[c];
                }
            }
        }

        // 创建输入 tensor
        std::vector<int64_t> input_shape = {1, 3, input.rows, input.cols};
        Ort::MemoryInfo memory_info = Ort::MemoryInfo::CreateCpu(OrtArenaAllocator, OrtMemTypeDefault);
        Ort::Value input_ort = Ort::Value::CreateTensor<float>(
            memory_info, input_tensor.data(), input_tensor.size(),
            input_shape.data(), input_shape.size()
        );

        // 推理
        const char* input_names[] = {"input"};
        const char* output_names[] = {"output"};
        auto output_tensors = session_.Run(Ort::RunOptions{nullptr},
            input_names, &input_ort, 1, output_names, 1);

        // 后处理
        float* output_data = output_tensors[0].GetTensorMutableData<float>();
        cv::Mat output(input.rows, input.cols, CV_32FC3);

        for (int c = 0; c < 3; ++c) {
            for (int h = 0; h < input.rows; ++h) {
                for (int w = 0; w < input.cols; ++w) {
                    output.at<cv::Vec3f>(h, w)[c] =
                        output_data[c * input.rows * input.cols + h * input.cols + w];
                }
            }
        }

        cv::cvtColor(output, output, cv::COLOR_RGB2BGR);
        output.convertTo(output, CV_8UC3, 255.0);

        return output;
    }

private:
    Ort::Env env_;
    Ort::SessionOptions session_options_;
    Ort::Session session_{nullptr};
};

int main() {
    DnCNNDenoiser denoiser("dncnn.onnx");

    cv::Mat noisy = cv::imread("noisy.jpg");
    cv::Mat denoised = denoiser.denoise(noisy);
    cv::imwrite("denoised.jpg", denoised);

    return 0;
}
```

```python
    mse = np.mean((img1 - img2) ** 2)
    return 10 * np.log10(max_val ** 2 / mse)
```

### 7.2 测试脚本

```python
# evaluate.py
import torch
import numpy as np
import cv2
from models.dncnn import DnCNN
from skimage.metrics import peak_signal_noise_ratio, structural_similarity
import glob

def evaluate(model, test_images, noise_level=25, device='cuda'):
    model.eval()
    psnr_list = []
    ssim_list = []

    with torch.no_grad():
        for img_path in test_images:
            # 读取干净图像
            clean = cv2.imread(img_path)
            clean = cv2.cvtColor(clean, cv2.COLOR_BGR2RGB)

            # 添加噪声
            noise = np.random.randn(*clean.shape) * noise_level
            noisy = np.clip(clean + noise, 0, 255).astype(np.uint8)

            # 推理
            noisy_tensor = torch.from_numpy(noisy.astype(np.float32) / 255.0)
            noisy_tensor = noisy_tensor.permute(2, 0, 1).unsqueeze(0).to(device)
            output = model(noisy_tensor)
            output = output.squeeze().permute(1, 2, 0).cpu().numpy()
            output = np.clip(output * 255, 0, 255).astype(np.uint8)

            # 计算指标
            psnr = peak_signal_noise_ratio(clean, output)
            ssim_val = structural_similarity(clean, output, channel_axis=2)

            psnr_list.append(psnr)
            ssim_list.append(ssim_val)

    print(f'Average PSNR: {np.mean(psnr_list):.2f} dB')
    print(f'Average SSIM: {np.mean(ssim_list):.4f}')

    return np.mean(psnr_list), np.mean(ssim_list)
```

## 8. 面试要点

### 高频问题

**Q1: 讲一下 DnCNN 的网络结构和设计思路**

**回答**：
- 采用简单的 VGG 风格网络：Conv-BN-ReLU 堆叠
- 使用残差学习：预测噪声而非干净图像
- 17 层深度，64 个特征通道
- 不使用池化，保持分辨率

**Q2: 为什么使用残差学习？**

**回答**：
- 噪声比图像内容更简单，更容易学习
- 梯度传播更稳定，训练更快收敛
- 当噪声很小时，输出接近输入（恒等映射）

**Q3: 如何将模型部署到移动端？**

**回答**：
1. **模型导出**：PyTorch → ONNX
2. **模型优化**：
   - 量化（FP32 → FP16/INT8）
   - 剪枝（移除不重要的通道）
3. **推理框架**：
   - ONNX Runtime
   - NCNN（腾讯开源，移动端优化）
   - MNN（阿里开源）
4. **硬件加速**：利用 NPU

**Q4: 真实噪声和合成噪声有什么区别？如何处理真实噪声？**

**回答**：
- 合成噪声通常是加性高斯白噪声，真实噪声更复杂
- 真实噪声包含泊松噪声、读出噪声、暗电流等
- 处理方法：
  - 使用真实噪声数据集（SIDD、DND）训练
  - 使用更复杂的噪声模型进行数据增强
  - 使用噪声估计网络先估计噪声等级
