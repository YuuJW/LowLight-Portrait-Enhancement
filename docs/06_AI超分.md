## 1. 超分辨率概述

### 1.1 什么是超分辨率？

超分辨率（Super-Resolution, SR）是从低分辨率图像恢复高分辨率图像的技术。

```
┌─────────────┐         ┌─────────────────────────┐
│  LR 图像    │  ──SR──▶│      HR 图像            │
│  100×100    │         │      400×400            │
└─────────────┘         └─────────────────────────┘
```

### 1.2 应用场景

| 场景 | 应用 |
|------|------|
| 手机数码变焦 | 增强远距离拍摄清晰度 |
| 老照片修复 | 提升老照片分辨率 |
| 视频增强 | 将 720p 视频升级到 4K |
| 医学影像 | 增强 CT/MRI 图像细节 |

### 1.3 超分的挑战

- **病态问题**：一个 LR 可能对应多个 HR
- **感知质量 vs 保真度**：过度锐化 vs 过度平滑
- **真实退化复杂**：模糊、噪声、压缩等混合退化

### 1.4 退化模型

**经典退化模型**：
```
LR = (HR * k) ↓s + n
其中：
  k = 模糊核
  ↓s = 下采样（scale s）
  n = 噪声
```

**真实退化模型（Real-ESRGAN）**：
```
LR = [(HR * k1) ↓s1 + n1] * k2 ↓s2 + n2 + JPEG压缩
```

## 2. 经典模型介绍

### 2.1 模型演进

```
SRCNN (2014) - 首个深度学习超分
    ↓
VDSR (2016) - 更深网络 + 残差学习
    ↓
EDSR (2017) - 去除 BN，更大模型
    ↓
RCAN (2018) - 通道注意力
    ↓
ESRGAN (2018) - GAN 提升感知质量
    ↓
Real-ESRGAN (2021) - 处理真实退化
    ↓
SwinIR (2021) - Transformer 架构
```

### 2.2 ESRGAN vs Real-ESRGAN

| 特点 | ESRGAN | Real-ESRGAN |
|------|--------|-------------|
| 训练数据 | 合成退化 | 真实世界退化 |
| 退化模型 | 简单 bicubic | 复杂高阶退化 |
| 应用场景 | 干净图像 | 真实场景图像 |
| 鲁棒性 | 一般 | 强 |

### 2.3 RRDB 网络结构

Real-ESRGAN 的生成器使用 RRDB（Residual-in-Residual Dense Block）：

```
Input
  ↓
Conv(3×3, 64)
  ↓
[RRDB] × 23
  ↓
Conv(3×3, 64) + Residual
  ↓
Upsample (PixelShuffle)
  ↓
Conv(3×3, 64)
  ↓
Conv(3×3, 3)
  ↓
Output
```

### 3.3 Python API 使用

```python
# super_resolution.py
import cv2
import numpy as np
from basicsr.archs.rrdbnet_arch import RRDBNet
from realesrgan import RealESRGANer

def create_realesrgan(model_name='RealESRGAN_x4plus', scale=4, device='cuda'):
    """
    创建 Real-ESRGAN 模型

    Args:
        model_name: 模型名称
        scale: 放大倍数
        device: 设备

    Returns:
        upsampler: RealESRGANer 实例
    """
    if model_name == 'RealESRGAN_x4plus':
        model = RRDBNet(num_in_ch=3, num_out_ch=3, num_feat=64, num_block=23, num_grow_ch=32, scale=4)
        model_path = 'weights/RealESRGAN_x4plus.pth'
    elif model_name == 'RealESRGAN_x4plus_anime_6B':
        model = RRDBNet(num_in_ch=3, num_out_ch=3, num_feat=64, num_block=6, num_grow_ch=32, scale=4)
        model_path = 'weights/RealESRGAN_x4plus_anime_6B.pth'
    elif model_name == 'RealESRGAN_x2plus':
        model = RRDBNet(num_in_ch=3, num_out_ch=3, num_feat=64, num_block=23, num_grow_ch=32, scale=2)
        model_path = 'weights/RealESRGAN_x2plus.pth'

    upsampler = RealESRGANer(
        scale=scale,
        model_path=model_path,
        model=model,
        tile=0,  # 分块处理，0 表示不分块
        tile_pad=10,
        pre_pad=0,
        half=True,  # FP16 加速
        device=device
    )

    return upsampler


def super_resolve(image_path, output_path, upsampler, outscale=4):
    """
    图像超分辨率

    Args:
        image_path: 输入图像路径
        output_path: 输出图像路径
        upsampler: RealESRGANer 实例
        outscale: 输出放大倍数
    """
    img = cv2.imread(image_path, cv2.IMREAD_UNCHANGED)

    # 超分处理
    output, _ = upsampler.enhance(img, outscale=outscale)

    # 保存结果
    cv2.imwrite(output_path, output)
    print(f'Saved to {output_path}')


if __name__ == '__main__':
    upsampler = create_realesrgan('RealESRGAN_x4plus')
    super_resolve('input.jpg', 'output.jpg', upsampler)
```

### 3.4 分块处理大图

```python
def super_resolve_large(image_path, output_path, upsampler, tile_size=512, outscale=4):
    """
    分块处理大图（避免显存溢出）
    """
    img = cv2.imread(image_path, cv2.IMREAD_UNCHANGED)
    h, w = img.shape[:2]

    # 计算输出尺寸
    out_h, out_w = h * outscale, w * outscale
    output = np.zeros((out_h, out_w, 3), dtype=np.uint8)

    # 分块处理
    tile_pad = 10
    for y in range(0, h, tile_size):
        for x in range(0, w, tile_size):
            # 计算块边界（加 padding）
            y1 = max(0, y - tile_pad)
            x1 = max(0, x - tile_pad)
            y2 = min(h, y + tile_size + tile_pad)
            x2 = min(w, x + tile_size + tile_pad)

            # 提取块
            tile = img[y1:y2, x1:x2]

            # 超分
            tile_output, _ = upsampler.enhance(tile, outscale=outscale)

            # 计算输出位置（去除 padding）
            out_y1 = (y - y1) * outscale
            out_x1 = (x - x1) * outscale
            out_y2 = out_y1 + min(tile_size, h - y) * outscale
            out_x2 = out_x1 + min(tile_size, w - x) * outscale

            # 放入输出图像
            output[y*outscale:(y+tile_size)*outscale, x*outscale:(x+tile_size)*outscale] = \
                tile_output[out_y1:out_y2, out_x1:out_x2]

    cv2.imwrite(output_path, output)
```

## 4. 模型轻量化

### 4.1 为什么需要轻量化？

| 模型 | 参数量 | 推理时间（512×512） |
|------|--------|-------------------|
| RealESRGAN_x4plus | 16.7M | ~300ms (GPU) |
| RealESRGAN_x4plus_anime_6B | 1.5M | ~50ms (GPU) |
| 自定义轻量版 | <1M | <30ms (GPU) |

### 4.2 轻量化方法

#### 方法1：使用更小的模型

```python

```

#### 方法2：模型量化

```python
# FP16 量化（推理时）
upsampler = RealESRGANer(
    ...,
    half=True  # 使用 FP16
)
```

#### 方法3：知识蒸馏

```python
# 蒸馏训练示例
class DistillationLoss(nn.Module):
    def __init__(self, teacher_model, alpha=0.5, temperature=4):
        super().__init__()
        self.teacher = teacher_model
        self.teacher.eval()
        self.alpha = alpha
        self.temperature = temperature
        self.mse = nn.MSELoss()
        self.kl = nn.KLDivLoss(reduction='batchmean')

    def forward(self, student_output, target, student_input):
        # 像素损失
        pixel_loss = self.mse(student_output, target)

        # 蒸馏损失
        with torch.no_grad():
            teacher_output = self.teacher(student_input)
        distill_loss = self.mse(student_output, teacher_output)

        return pixel_loss + self.alpha * distill_loss
```

### 4.3 ONNX 导出

```python
# export_realesrgan_onnx.py
import torch
from basicsr.archs.rrdbnet_arch import RRDBNet

def export_onnx(model_path, onnx_path, scale=4, num_block=6):
    # 创建模型
    model = RRDBNet(num_in_ch=3, num_out_ch=3, num_feat=64, num_block=num_block, num_grow_ch=32, scale=scale)

    # 加载权重
    state_dict = torch.load(model_path, map_location='cpu')
    if 'params_ema' in state_dict:
        state_dict = state_dict['params_ema']
    elif 'params' in state_dict:
        state_dict = state_dict['params']
    model.load_state_dict(state_dict, strict=True)
    model.eval()

    # 导出 ONNX
    dummy_input = torch.randn(1, 3, 64, 64)
    torch.onnx.export(
        model,
        dummy_input,
        onnx_path,
        input_names=['input'],
        output_names=['output'],
        dynamic_axes={
            'input': {0: 'batch', 2: 'height', 3: 'width'},
            'output': {0: 'batch', 2: 'height', 3: 'width'}
        },
        opset_version=11
    )
    print(f'Exported to {onnx_path}')


if __name__ == '__main__':
    export_onnx(
        'weights/RealESRGAN_x4plus_anime_6B.pth',
        'weights/realesrgan_anime.onnx',
        scale=4,
        num_block=6
    )
```

## 5. 模型部署

### 5.1 ONNX Runtime 推理

```python
# inference_onnx_sr.py
import onnxruntime as ort
import numpy as np
import cv2

class RealESRGANONNX:
    def __init__(self, onnx_path, device='cuda'):
        providers = ['CUDAExecutionProvider', 'CPUExecutionProvider'] if device == 'cuda' else ['CPUExecutionProvider']
        self.session = ort.InferenceSession(onnx_path, providers=providers)
        self.input_name = self.session.get_inputs()[0].name

    def enhance(self, img, outscale=4):
        """
        图像超分

        Args:
            img: BGR 格式图像 (H, W, 3)
            outscale: 输出放大倍数

        Returns:
            output: 超分后的图像
        """
        # 预处理
        img_float = img.astype(np.float32) / 255.0
        img_rgb = cv2.cvtColor(img_float, cv2.COLOR_BGR2RGB)
        img_tensor = img_rgb.transpose(2, 0, 1)[np.newaxis, ...]

        # 推理
        output = self.session.run(None, {self.input_name: img_tensor})[0]

        # 后处理
        output = output.squeeze().transpose(1, 2, 0)
        output = cv2.cvtColor(output, cv2.COLOR_RGB2BGR)
        output = np.clip(output * 255, 0, 255).astype(np.uint8)

        return output


if __name__ == '__main__':
    sr = RealESRGANONNX('weights/realesrgan_anime.onnx')
    img = cv2.imread('input.jpg')
    output = sr.enhance(img)
    cv2.imwrite('output_sr.jpg', output)
```

## 6. 效果评估

### 6.1 评估指标

**PSNR / SSIM**：衡量保真度
- 越高越好
- 但不一定代表感知质量

**LPIPS**：衡量感知质量
- 越低越好
- 更符合人眼感知

**NIQE / BRISQUE**：无参考质量评估
- 不需要 GT
- 适合真实场景

### 6.2 测试集

| 数据集 | 描述 | 用途 |
|--------|------|------|
| Set5 | 5 张经典图像 | 快速测试 |
| Set14 | 14 张图像 | 标准测试 |
| BSD100 | 100 张图像 | 综合测试 |
| Urban100 | 100 张城市场景 | 测试细节恢复 |

### 6.3 评估脚本

```python
# evaluate_sr.py
import cv2
import numpy as np
from skimage.metrics import peak_signal_noise_ratio, structural_similarity
import lpips
import torch
import glob

def evaluate_sr(sr_folder, gt_folder, scale=4):
    """
    评估超分效果
    """
    psnr_list = []
    ssim_list = []

    # LPIPS
    loss_fn = lpips.LPIPS(net='alex')

    sr_images = sorted(glob.glob(f'{sr_folder}/*.png'))
    gt_images = sorted(glob.glob(f'{gt_folder}/*.png'))

    for sr_path, gt_path in zip(sr_images, gt_images):
        sr = cv2.imread(sr_path)
        gt = cv2.imread(gt_path)

        # 确保尺寸匹配
        sr = cv2.resize(sr, (gt.shape[1], gt.shape[0]))

        # PSNR
        psnr = peak_signal_noise_ratio(gt, sr)
        psnr_list.append(psnr)

        # SSIM
        ssim = structural_similarity(gt, sr, channel_axis=2)
        ssim_list.append(ssim)

    print(f'Average PSNR: {np.mean(psnr_list):.2f} dB')
    print(f'Average SSIM: {np.mean(ssim_list):.4f}')


if __name__ == '__main__':
    evaluate_sr('results/sr', 'datasets/Set5/HR')
```

## 7. 面试要点

### 高频问题

**Q1: 图像超分辨率的主要方法有哪些？**

**回答**：
- **传统方法**：双三次插值、基于稀疏表示
- **深度学习方法**：
  - SRCNN：首个端到端 CNN
  - EDSR：去除 BN，更深更宽
  - RCAN：通道注意力机制
  - ESRGAN/Real-ESRGAN：GAN 提升感知质量
  - SwinIR：Transformer 架构

**Q2: ESRGAN 和 Real-ESRGAN 的区别是什么？**

**回答**：
- **退化模型**：ESRGAN 使用简单 bicubic，Real-ESRGAN 使用复杂高阶退化
- **训练数据**：Real-ESRGAN 使用真实退化模拟
- **鲁棒性**：Real-ESRGAN 对真实场景更鲁棒
- **应用场景**：Real-ESRGAN 更适合真实照片

**Q3: 如何平衡超分的保真度和感知质量？**

**回答**：
- **保真度（PSNR）**：使用 L1/L2 损失
- **感知质量（LPIPS）**：使用感知损失 + GAN 损失
- **权衡方法**：
  - 调整损失权重
  - 使用 ESRGAN 的网络插值技术
  - 后处理调整锐度

**Q4: 如何将超分模型部署到移动端？**

**回答**：
1. **模型选择**：使用轻量版（如 anime_6B）
2. **量化**：FP16 或 INT8
3. **导出**：PyTorch → ONNX → NCNN/MNN
4. **优化**：
   - 分块处理大图
   - 使用 NPU 加速
   - 降低输出分辨率

**Q5: 超分在手机影像中有什么应用？**

**回答**：
- **数码变焦增强**：远摄时提升清晰度
- **多摄融合**：融合主摄和长焦细节
- **夜景增强**：低光图像细节恢复
- **照片编辑**：裁剪后恢复分辨率
